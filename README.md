# Java Console Calculator

A simple and interactive console-based calculator built using Java.  
The application allows users to perform basic arithmetic operations using a menu-driven approach.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulo
- Menu-driven interface
- Loop-based execution
- Division by zero handling

## Technologies Used
- Java
- Scanner class
- Switch-case
- Loops

## How to Run
1. Compile the program:
   javac Calculator.java
2. Run the program:
   java Calculator
